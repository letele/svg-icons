# svg-icons &#10024;
svg icons for react applications 

## Installation &#128640;
<code>npm install @letele/svg-icons</code>

## Usage &#129335;
```js
import  { Android } from  "@letele/svg-icons" 

function App() {
    return (
        <Android />
    )
}
```
